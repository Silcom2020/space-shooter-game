## 🎮 ゲーム概要

Space Shooterは、4つの異なるステージで様々な敵と戦う本格的な宇宙シューティングゲームです。パワーアップシステム、コンボシステム、動的背景オブジェクトなど、豊富な機能を搭載しています。

### 🌟 主な特徴

- **4つのユニークなステージ** - 宇宙空間、惑星軌道、小惑星帯、敵基地
- **多様な敵タイプ** - 基本敵、高速敵、分裂敵、追尾敵、ボス、巨大ボス
- **パワーアップシステム** - 連射強化、パワーショット、シールド、ライフ回復
- **コンボシステム** - 連続撃破でスコアボーナス
- **動的背景オブジェクト** - 流れる惑星、回転する小惑星、射撃するタレット
- **本格的UI/UX** - タイトル画面、一時停止、設定メニュー、ヘルプシステム
- **パフォーマンス最適化** - 60FPS安定動作、メモリ効率化

## 🎯 ゲームプレイ

### 基本操作
- **↑↓←→** - 戦闘機の移動
- **SPACE** - レーザー射撃
- **P** - 一時停止/再開
- **S** - ステージ選択
- **1-4** - 直接ステージ選択
- **H** - ヘルプ表示
- **ESC** - メインメニューに戻る
- **R** - ゲームリスタート

### ゲームシステム

#### 🎪 ステージシステム
1. **ステージ1: 宇宙空間** - 基本的な宇宙戦闘
2. **ステージ2: 惑星軌道** - 人工衛星を避けながらの戦闘（敵出現率+20%）
3. **ステージ3: 小惑星帯** - 回転する小惑星を縫って進む（敵出現率+50%）
4. **ステージ4: 敵基地** - 基地防衛システムとの総力戦（敵出現率+100%）

#### 👾 敵タイプ
- **基本敵** (10点) - 標準的な敵
- **高速敵** (15点) - 素早く移動、急降下攻撃
- **分裂敵** (25点) - 撃破時に2体の高速敵に分裂
- **追尾敵** (30点) - プレイヤーを狙って弾を撃つ
- **ボス** (100点) - 3-5方向射撃の強敵
- **巨大ボス** (250点) - 7方向射撃の最強敵

#### 💎 パワーアップアイテム
- **🟢 連射強化** - 弾の発射間隔を短縮（10秒間）
- **🟠 パワーショット** - 3方向同時射撃（10秒間）
- **🔵 シールド** - 敵の攻撃を無効化（5秒間）
- **🔴 ライフ回復** - ライフを1つ回復

#### 🎯 スコアシステム
- **コンボシステム** - 3秒以内の連続撃破でコンボ継続
- **多重ボーナス** - コンボ×ステージ×レベルの掛け算
- **ライフボーナス** - 10,000点ごとにライフ獲得
- **レベルアップボーナス** - レベル×50点

#### 🌌 動的背景オブジェクト
- **惑星軌道**: 大型惑星、揺れ動く人工衛星（衝突でライフ減少）
- **小惑星帯**: 回転する小惑星、高速宇宙塵（衝突でライフ減少）
- **敵基地**: 点滅する構造物、プレイヤー狙撃タレット（衝突でライフ減少）
